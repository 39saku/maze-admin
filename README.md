# Maze Admin Server (運営管理システム)

Maze Challengeの運営・管理用サーバーです。
参加者のスコアをリアルタイムで集計し、ランキング表示やバトルモードの実行を行います。

## ⚠️ 注意（管理者へ）
- 参加者のニックネームやソースコードが `scores.json` に保存されます。

## 📁 フォルダ構成
- `server.js`: Node.js(Express + Socket.io) サーバー本体
- `public/`: 運営用画面（ランキング・バトルモード）
- `data/`: 参加者と共有する迷路データ (`maps.js`)
- `scores.json`: 参加者の提出データ（自動生成）
